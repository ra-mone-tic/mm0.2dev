# MeowMap

Интерактивная карта культурных событий Калининграда. Демо-версия сервиса для отображения мероприятий с геолокацией

## Демо

https://ra-mone-tic.github.io/mm0.2dev/

## Технологии

- Frontend: Vanilla JavaScript (ES6 Modules), MapLibre GL
- Backend: Python ETL (Google Sheets → JSON)
- Геокодирование: Каскад ArcGIS/Yandex/Nominatim
- Автоматическое обновление: GitHub Actions
- Стилизация: CSS Variables, адаптивный дизайн

## Функциональность

- Интерактивная карта с кластеризацией маркеров
- Поиск и фильтры по названию и локации
- Календарь с навигацией по датам
- Темы оформления (светлая/темная/)
- Адаптивный дизайн для всех устройств
- Доступность (ARIA, keyboard navigation)
- Поделиться событием с прямой ссылкой

## Архитектура

```
assets/js/
├── main.js          # Точка входа, инициализация модулей
├── map.js           # MapLibre карта + маркеры
├── event-list.js    # Список событий + сайдбар
├── search.js        # Поиск + результаты
├── calendar.js      # Календарь + навигация
├── utils.js         # Утилиты (геокодинг, санитизация)
├── constants.js     # Конфигурация + селекторы
└── theme.js         # Переключение тем
```

Особенности архитектуры:
- Модульная структура без зависимостей
- Event-driven коммуникация между модулями
- Асинхронная загрузка данных
- Graceful error handling

## Для разработчиков

Требования:
- Python 3.8+ для ETL скрипта
- Google Sheets URL для данных событий
- Геокодинг API ключи (Yandex API ключ для лучшего качества)

ETL процесс (автоматический):
1. Извлечение данных из Google Sheets (CSV экспорт)
2. Парсинг и нормализация событий (дата, заголовок, место, описание)
3. Геокодирование адресов с каскадным fallback (ArcGIS → Yandex → Nominatim)
4. Валидация, дедупликация и сохранение в events.json
5. Автоматическое обновление через GitHub Actions каждый час

Развертывание:
- Статический сайт на GitHub Pages
- Автоматическое обновление данных через CI/CD pipeline
